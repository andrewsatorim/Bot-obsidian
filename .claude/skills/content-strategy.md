# Content Strategy

## Content Planning Framework

### 1. Define the Goal
| Goal | Content type | Metric |
|------|-------------|--------|
| **Awareness** | Blog, social, video | Traffic, impressions |
| **Consideration** | Comparisons, case studies | Time on page, signups |
| **Conversion** | Landing pages, demos | Trial starts, purchases |
| **Retention** | Docs, tutorials, changelog | Engagement, NPS |
| **Advocacy** | Community, UGC | Referrals, mentions |

### 2. Know the Audience
```
Persona: [Name]
Role: [Job title / situation]
Pain: [What frustrates them daily]
Goal: [What they want to achieve]
Objection: [Why they hesitate]
Channels: [Where they consume content]
Search terms: [What they Google]
```

### 3. Content Audit
- What exists? → catalog all content
- What performs? → top 20% by traffic/conversions
- What's outdated? → update or remove
- What's missing? → gaps in the buyer journey

## Blog Post Structure

### SEO-Optimized Article Template
```markdown
# [H1: Primary keyword + benefit] (50-60 chars)

[Opening hook — 2-3 sentences addressing the reader's problem]

[Brief overview of what this post covers]

## [H2: First major section]
[Content with practical examples]
[Include data, screenshots, or code]

## [H2: Second major section]
[Content]

### [H3: Subsection if needed]
[Detailed breakdown]

## [H2: Third major section]
[Content]

## Key Takeaways
- Bullet 1
- Bullet 2
- Bullet 3

## FAQ (optional — targets "People also ask")
### [Question with long-tail keyword]?
[Concise answer]

---
[Author bio] | [Related posts] | [CTA]
```

### Article Types by Intent

| Search intent | Article type | Example |
|---------------|-------------|---------|
| **Informational** | How-to, guide, explainer | "How to set up CI/CD with GitHub Actions" |
| **Commercial** | Comparison, review, "best of" | "Top 5 headless CMS for Next.js in 2026" |
| **Navigational** | Documentation, reference | "React useEffect — complete guide" |
| **Transactional** | Landing page, pricing | "Pricing — start free, scale as you grow" |

## SEO Content Workflow

1. **Keyword research** — find topics people search for
   - Seed keywords → expand with "People also ask", autocomplete
   - Check volume (>100/month) and difficulty
   - Target long-tail keywords for new sites
   
2. **Outline** — structure before writing
   - Check top 5 results for the keyword
   - Note what they cover and what they miss
   - Plan to be more complete, practical, or current

3. **Write** — quality over quantity
   - Lead with value (no long introductions)
   - Use examples, screenshots, code blocks
   - Internal link to related content
   - External link to authoritative sources

4. **Optimize**
   - Primary keyword in: title, H1, first paragraph, URL, meta description
   - Related keywords naturally throughout
   - Alt text on images with relevant keywords
   - Schema markup (FAQ, HowTo, Article)

5. **Promote**
   - Share on social (3+ times with different angles)
   - Send to newsletter subscribers
   - Cross-post summary on LinkedIn/Twitter
   - Reach out for backlinks from related content

## Content Calendar

### Monthly Planning Template
```
Week 1: [Pillar content] — long-form guide or tutorial
Week 2: [Supporting content] — related shorter post
Week 3: [Engagement content] — opinion, case study, comparison
Week 4: [Update/repurpose] — refresh old content or compile highlights
```

### Cadence Recommendations
| Stage | Blog | Newsletter | Social |
|-------|------|-----------|--------|
| Starting | 1/week | 1/week | 3-5/week |
| Growing | 2/week | 1/week | daily |
| Established | 3-5/week | 2/week | 2-3x/day |

## Content Repurposing

```
Blog post (2,000 words)
├── Twitter thread (key points)
├── LinkedIn post (main insight)
├── Instagram carousel (visual tips)
├── Newsletter section (summary + link)
├── YouTube video (walkthrough)
├── Podcast talking points
└── Documentation page (if technical)
```

One piece of content → 7+ distribution formats.

## Measuring Content Performance

| Metric | Indicates | Action |
|--------|-----------|--------|
| Organic traffic | SEO working | Double down on similar topics |
| Time on page | Content quality | Improve engaging sections |
| Bounce rate | Relevance match | Better titles/intros |
| Conversions | Business value | Optimize CTAs |
| Backlinks | Authority | Promote more aggressively |
| Social shares | Resonance | Create more like this |
