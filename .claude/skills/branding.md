# Branding

## Brand Positioning

### Positioning Statement Template
```
For [target audience]
who [need/problem],
[Product] is a [category]
that [key benefit].
Unlike [competitor/alternative],
we [unique differentiator].
```

### Example
```
For indie developers
who waste hours on deployment,
ShipFast is a deployment platform
that gets your app live in one command.
Unlike AWS or manual setups,
we handle infrastructure so you can focus on code.
```

## Messaging Framework

### Hierarchy
```
Mission (why you exist)
  → Vision (where you're heading)
    → Value proposition (what you offer)
      → Key messages (3 pillars)
        → Proof points (evidence for each)
          → Tagline (memorable summary)
```

### Template
| Layer | Content |
|-------|---------|
| **Mission** | "We exist to [verb] [for whom] by [how]" |
| **Vision** | "A world where [ideal future state]" |
| **Value prop** | "[Product] helps [audience] [achieve outcome] by [method]" |
| **Pillar 1** | [Benefit] — supported by [proof] |
| **Pillar 2** | [Benefit] — supported by [proof] |
| **Pillar 3** | [Benefit] — supported by [proof] |
| **Tagline** | [3-7 words capturing the essence] |

### Tagline Formulas
- **Command:** "Just do it" / "Think different"
- **Benefit:** "Save time. Ship faster."
- **Metaphor:** "Your AI co-pilot for code"
- **Question:** "What would you build?"
- **Provocative:** "Deployment shouldn't take a PhD"

## Voice & Tone

### Voice (consistent personality)

Define 3-4 voice attributes with a spectrum:

| We are | We are NOT |
|--------|------------|
| Confident | Arrogant |
| Friendly | Childish |
| Clear | Dumbed-down |
| Bold | Reckless |

### Tone (adapts to context)

| Context | Tone shift | Example |
|---------|-----------|---------|
| Homepage | Confident + aspirational | "Build the future of finance" |
| Docs | Clear + neutral | "Pass your API key as a header" |
| Error | Calm + helpful | "We couldn't save your changes. Try again?" |
| Changelog | Excited + concise | "Dark mode is here!" |
| Support | Empathetic + solution-focused | "I understand the frustration. Here's what we can do" |
| Legal | Precise + trustworthy | "We never sell your personal data" |

### Writing Style Guide

**Always:**
- Use active voice
- Write in present tense
- Address the user as "you"
- Use contractions (we're, you'll, it's)
- Be specific with numbers and outcomes

**Never:**
- Use jargon without explaining it
- Use "utilize" (say "use")
- Use "leverage" as a verb (say "use")
- Start with "Welcome to..." (boring)
- Say "sorry for the inconvenience" (empty)
- Use exclamation marks excessively!!!

## Brand Name Checklist

- [ ] Easy to spell and pronounce
- [ ] Domain available (.com or relevant TLD)
- [ ] Social handles available (or buyable)
- [ ] Not trademarked in your category
- [ ] No negative meanings in other languages
- [ ] Works as a verb: "Just [brand] it"
- [ ] Looks good in a logo/favicon
- [ ] Memorable after hearing it once

## Visual Brand Quick Reference

| Element | Guideline |
|---------|-----------|
| **Logo** | Works at 16px (favicon) to billboard size |
| **Colors** | 1 primary, 1-2 secondary, 1 neutral scale, semantic colors |
| **Typography** | 1 display font (headings), 1 body font (text) |
| **Imagery** | Consistent style: photography OR illustration, not mixed |
| **Iconography** | One icon set, consistent weight and style |
| **Spacing** | Based on 4/8px grid system |

## Brand Audit Questions

When reviewing any touchpoint, ask:
1. **Would a stranger know what we do** in 5 seconds?
2. **Does it feel like us?** — consistent voice, visuals, tone?
3. **Is the value clear?** — benefit to the user, not feature list?
4. **Would a user trust this?** — professional, honest, credible?
5. **Is there one clear next step?** — CTA is obvious?
